# 主要資産クラス 100年パフォーマンス比較チャート

1924年から2024年までの主要資産クラスの長期パフォーマンスを比較するインタラクティブなWebアプリケーション。

## 特徴

- **6つの主要資産クラス**を比較
  - S&P 500（米国株式）
  - 金（ゴールド）
  - 米国10年債
  - 日経平均株価
  - 米ドル指数
  - NASDAQ 100

- **インタラクティブなチャート**
  - Chart.jsを使用した対数スケールの折れ線グラフ
  - チェックボックスで表示/非表示を切り替え可能
  - ツールチップでデータの詳細を確認

- **投資シミュレーター**
  - 100万円を任意の期間投資した場合の結果を計算
  - 最終価値、総リターン、年率リターンを表示
  - 投資額の推移をグラフで視覚化

- **詳細な統計データ**
  - 各資産の100年間総リターン
  - 年平均リターン
  - 10年ごとの期間別リターン比較表

## ローカルでの実行

1. リポジトリをクローン
```bash
git clone https://github.com/yourusername/stock.chart.git
cd stock.chart
```

2. ブラウザで `index.html` を開く

## Vercelでのデプロイ

このプロジェクトはVercelに簡単にデプロイできます。

1. GitHubリポジトリと連携
2. Vercelダッシュボードから「New Project」を選択
3. このリポジトリをインポート
4. 自動的にデプロイされます
