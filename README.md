# 主要資産 パフォーマンス比較チャート

1924年から2024年までの主要資産クラスの長期パフォーマンスを比較するインタラクティブなWebアプリケーション。

## 特徴

### 📊 8つの主要資産クラスを比較
- S&P 500（米国株式）
- 金（ゴールド）
- 米国10年債
- 日経平均株価
- 米ドル指数
- NASDAQ 100
- ビットコイン
- 日本円

### 🎯 期間選択機能
- **6つの期間で比較可能**: 100年、50年、20年、10年、5年、1年
- 各期間で全資産が**¥100万円スタート**に正規化
- 「同時に同額投資したら」の比較が可能

### 📈 インタラクティブなチャート
- Chart.jsを使用した対数スケールの折れ線グラフ
- チェックボックスで資産の表示/非表示を切り替え
- **動的Y軸調整**: データ範囲に応じて自動で最適表示
- ツールチップで詳細データを確認（¥単位、千円/万円/億円表記）
- タップ外で閉じる（モバイル対応）

### 💰 投資シミュレーター
- 100万円を任意の期間投資した場合の結果を計算
- 過去データと未来予測（年平均リターン使用）に対応
- 最終価値、総リターン、年率リターンを表示
- 投資額の推移をグラフで視覚化
- 実線（過去データ）と点線（予測）で区別

### 📊 詳細な統計データ
- 各資産の長期総リターン
- 年平均リターン
- 10年ごとの期間別リターン比較表
- パフォーマンスランキング表示（一位、二位、三位）

### 📱 完全レスポンシブ対応
- デスクトップ、タブレット、スマートフォンで最適表示
- モバイルでは凡例がチャート下に移動
- タッチ操作に最適化

## 主な機能のハイライト

### 期間別正規化比較
各期間（100年/50年/20年/10年/5年/1年）で、すべての資産を開始時点の¥100万円に正規化。異なる時期に開始された資産を公平に比較できます。

**例: 1年表示（2023→2024）**
- S&P 500: ¥100万円 → ¥132万円（+32%）
- ビットコイン: ¥100万円 → ¥177万円（+77%）
- 金: ¥100万円 → ¥135万円（+35%）
- 日本円: ¥100万円 → ¥97万円（-3%）

### 日本語に最適化
- 通貨: ¥（円）表記
- 単位: 千円、万円、億円（M/K表記なし）
- 完全日本語UI

## データソース

### 各資産の開始年と基準点
- **S&P 500**: 1924年〜（¥100基準）
- **金**: 1924年〜（¥100基準）
- **米国10年債**: 1924年〜（¥100基準）
- **日経平均**: 1949年〜（¥100基準、戦後算出開始）
- **日本円**: 1949年〜（¥100基準、対米ドル購買力）
- **米ドル指数**: 1973年〜（¥100基準、変動相場制開始）
- **NASDAQ 100**: 1985年〜（¥100基準）
- **ビットコイン**: 2010年〜（¥100基準）

### データの詳細度
- 1924-2000年: 主要年のみ
- 2000-2004年: 5年間隔
- 2005-2024年: **毎年のデータポイント**（短期比較に最適）

## ローカルでの実行

1. リポジトリをクローン
```bash
git clone https://github.com/hotaka0908/stock.git
cd stock
```

2. ブラウザで `index.html` を開く

## Vercelでのデプロイ

このプロジェクトはVercelに簡単にデプロイできます。

1. GitHubリポジトリと連携
2. Vercelダッシュボードから「New Project」を選択
3. このリポジトリをインポート
4. 自動的にデプロイされます
